## Detailed Polish Checklist

### Typography & Spacing
- [ ] Verify all heading sizes match original (H1, H2, H3)
- [ ] Check line-height and letter-spacing for readability
- [ ] Confirm padding/margins between sections
- [ ] Test font rendering on different browsers

### Colors & Contrast
- [ ] Green primary color matches original (#2D5016 or similar)
- [ ] Text contrast meets WCAG AA standards
- [ ] Hover states have sufficient color change
- [ ] Dark sections have proper text contrast

### Interactive Elements
- [ ] Button hover/active states working
- [ ] Links underline or change color on hover
- [ ] Form inputs have focus states
- [ ] Social media links open in new tabs

### Responsive Design
- [ ] Hero section stacks properly on mobile
- [ ] Service cards adapt to smaller screens (2-column on tablet, 1-column on mobile)
- [ ] Navigation collapses to hamburger menu on mobile
- [ ] Images scale appropriately without distortion
- [ ] Text remains readable at all breakpoints

### Performance & Loading
- [ ] Hero image loads quickly
- [ ] No layout shift as images load
- [ ] Smooth scrolling and transitions
- [ ] No console errors or warnings

### Accessibility
- [ ] All images have alt text
- [ ] Links have descriptive text (not "click here")
- [ ] Form labels associated with inputs
- [ ] Keyboard navigation works (Tab through elements)
- [ ] Focus indicators visible
